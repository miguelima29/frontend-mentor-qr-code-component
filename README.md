### What I learned

My main focus for this challenge was maintaining a solid, organized CSS structure. Using CSS custom properties allowed me to manage colors and sizing efficiently, keeping the code clean and scalable. I also reinforced my understanding of modern layout techniques by using Flexbox alongside the `gap` property to center and structure the card perfectly within the viewport. 

### Continued development

Moving forward, I want to keep refining my understanding of responsive layouts and scalable CSS architecture. I plan to take these foundational styling and structuring concepts and apply them to component-based workflows, particularly as I dive deeper into React development and build more complex interfaces.
